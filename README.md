## backend

* Prisma
* Fastify

-> Dev: <http://localhost:3333/api>
-> DB Admin: <http://localhost:5555/>

```bash
cd into backend
yarn
yarn dev
yarn generate
```

## web

* Next.js

-> Dev: <http://localhost:3000/>

```bash
cd into backend
yarn
yarn dev
yarn migrate
yarn generate
```

## mobile

* Expo
* Native Base <https://docs.nativebase.io>

```bash
cd into mobile
yarn
yarn start | android | ios
```
